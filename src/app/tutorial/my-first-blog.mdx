### 1. 下载注册游戏
<br />
Idleon这一款游戏可以在steam，ios，Android和网页游玩，而且四端数据互通。由于idleon在国内几乎是冷饭，所以国内并没有很多idleon的服务器。导致一些想要入坑的玩家无法进入游戏。所以我推荐一些进入游戏的方法。
<br />
电脑steam端可以直接搜索idleon并下载游玩，但是需要加速器，这里我推荐雷神加速器和奇游加速器，这两个加速器可以加速idleon的steam端。<br />
Ios可以在美区直接搜索idleon下载，需要梯子。Android可以在谷歌商店里面搜索idleon并下载，需要梯子。<br />
网页端链接：<a href="https://www.legendsofidleon.com/" target="_blank" rel="noopener noreferrer">https://www.legendsofidleon.com/</a>  <br />
电脑手机都可以用网页端游玩，同样需要梯子，网页端较为稳定。<br />
<br />
这里我推荐几款梯子（没有打广告的成分）：<br />
一款是ikuuu的梯子，ikuuu注册之后可以免费使用，每个月的流量几乎用不完。但是网络有时候不太稳定。<br />
<a href="https://ikuuu.club" target="_blank" rel="noopener noreferrer" style={{ color: '#2EA9DF' }}>https://ikuuu.club</a><br />
<a href="https://ikuuu.cc" target="_blank" rel="noopener noreferrer" style={{ color: '#2EA9DF' }}>https://ikuuu.cc</a><br />
<br />
另外一款是群友推荐给我的梯子：Sukuracat。这是一款付费梯子，在国内的服务器相对稳定。需要花一点钱买，我的建议是买那个巨量不限时流量包，38.8元260G，用完即止。Idleon只会在登陆界面和宠物抽奖界面强制联网，其他时候都不怎么消耗流量。你把idleon玩毕业了流量连一半都没用到。<br />
<a href="https://sakura-cat1.com/" target="_blank" rel="noopener noreferrer" style={{ color: '#2EA9DF' }}>https://sakura-cat1.com/</a>
<br />
游戏的注册有三种方式，ios，google账号和邮箱注册，这里我推荐google账号，因为google服务器相对稳定，登陆也快，如果你不知道怎么注册google账号的话可以在bilibili上搜索一下。Ios账号也可以注册，我就是用ios的，但是ios有时候不太稳定，会导致你登不进去。
还有一个邮箱注册，那个邮箱注册是你随便填一个邮箱都可以注册成功的，但是如果你要找回密码的时候就会给你的邮箱发邮件，此时如果你填了一个不存在的邮箱的话那你就找不回密码了。请慎重考虑。
<br />
Idleon有一个wiki百科，如果有什么不理解的地方建议先去wiki查，能解决很多问题：<a href="https://idleon.wiki/wiki/Main_Page" target="_blank" rel="noopener noreferrer" style={{ color: '#2EA9DF' }}>https://idleon.wiki/wiki/Main_Page</a> <br />
Idleon也有一个qq群：916573567 <br />
游戏作者是lavaflame2，群里有很多文件也能解决你的疑惑，建议大体先看一遍有什么。群内也有机器人，使用关键词即可召唤机器人发出对应内容。还有不懂的再问群友。<br />
Bilibili有一个叫阿曼阿曼阿曼慢的up主，他也讲了很多idleon的教程和指导什么的，也推荐去看。

### 2. 创建角色
<br />
好！注册完游戏你就开始创建角色了，点击create创建角色，右上角填你的名字，角色性别（角色性别感觉和没填一样）你的帽子和星座，星座你可以随便选一个，以后能够更改。选加经验的那个可以让你经验加的更快。<br />
<div style={{ color: '#E83015' }}>
请注意：你的角色名字不要太长，因为w2有相关任务就是要你丢出来你角色名字长度平方个的物品，如果角色名字太长你反而很难完成这个任务。<br />
其次后面还有一个任务是要一个P开头和S开头的角色讲一句话的任务，你可能需要找群友完成。如果你想自己一个人完成任务的话就建议你俩个角色一个是P开头一个是S开头
角色名字不能更改，角色职业也很难更改，不能删除角色，所以也请在取名和转职上慎重。
</div>
<br />
有一些辅助设置你需要注意，右下角菜单（menu）然后按设置（options）。以下是游戏辅助设置界面，推荐按照该图进行设置。<br />

### 3. 新手任务
<br />
你作为新手第一次进入游戏的时候，左上角有一个新手教程lava来带你度过新手期,他会要求你去做书卷蛇的任务。<br />
第一个任务：击杀绿蘑菇五只，地图右边有一个绿色的传送门，进去就是w1的第一张地图（后面以w+数字代指世界几）点击在跳动的东西(那个就是绿蘑菇)就可以攻击绿蘑菇，击杀五只之后左边的传送门回去，点书卷蛇提交任务。
<br />
需要知道的是，游戏里掉落的物品是不能直接自动拾取的，要鼠标左键点（或者鼠标按住左键拖拽）。如果你想要自动拾取的话可以在这里花钱购买。<br />
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/autoloot1.png" alt="自动拾取" style={{ width: '10%', position: 'center' }} />
  <img src="/images/tutorial/autoloot2.png" alt="自动拾取" style={{ width: '35%' }} /> <br />
</div>
花钱购买，差不多就35吧，游戏中特别刚需的礼包就是这个，其他的礼包如果不想充钱可以不买，但是自动拾取是非常建议要买的。
<br />
然后就是书卷蛇的第二个任务，书卷蛇会给你一个红线，你需要在商店里购买绳子。这个手指指着的就是w1的商店：
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/shopw1.png" alt="商店" style={{ width: '15%' }} />
</div>
下滑就能找到这个，按住物品然后拖拽到上方的购买框里面即可购买。
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/rope.png" alt="购买绳子" style={{ width: '15%' }} />
</div>
购买完按esc或者点UI(界面外面)的空白处关闭页面。然后点左边的打铁铺，点右边的拳套，再点左边的craft就做好了。最后关闭页面交任务。
<br />
补充一点，商店页面下方有一个时间，那个是商店补货的时间，也是你<span style={{ color: '#E83015' }}>每日重置的时间</span>，这个时间很重要。以后很多游戏刷新道具的时间都是这个。这个时间可以去调整，但这就是很后面的事情了。
<br />
当你打开打铁铺的时候，右上角有一个produce（生产页）里面可以自动造一些生产道具，按绿色的produce按钮即可制造。黄棕色的“Quick Deposit”按钮是快速放进箱子里的按钮。
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/produce.png" alt="生产" style={{ width: '20%' }} />
</div>
打开铁砧后，屏幕左下角会显示你的制造等级。
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/forgelv.png" alt="锻造等级" style={{ width: '30%' }} />
</div>
底部的两个按钮则是购买铁砧升级点数和花费升级点数，购买升级点数可以用游戏里的金钱货币来购买，也可以用一些怪物道具来购买。你的制造等级升级时也会给你升级点数。
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/produce-points.png" alt="锻造等级" style={{ width: '20%' }} />
</div>
点开花费点数的时候有三个加点选项，建议全加中间那个速度。因为经验在后期有更快的获取方法，而容量和你的材料袋的容量有关。加错也没事，能洗点，洗点成本也相当低。
顺带一提，虽然这个游戏的容错率非常高，但角色名字，创建角色，以及转职是无法改变的（宝石商店卖的转职卡只能改二转，一转和三转都不能改，改三转的道具只会在限时商店出现）。所以各位转职的时候请慎重。
<br />
回到正题，然后就是做书卷虫的下一个任务，他要你到达绿蘑菇的下一张地图：
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/map1w1.png" alt="w2" style={{ width: '40%' }} />
</div>
右边的传送门并不是刚开始就打开的，而是要达到一定击杀数才能打开，打开的时候是绿色。灰色传送门上面有开启要求。<br />
到达第二个地图之后，回去提交任务。别忘记打开背包，然后装备上你刚刚制作的拳套。可以拖拽到装备栏里对应格子，也可以直接鼠标对物品双击装备。
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/inventory.png" alt="背包" style={{ width: '15%' }} />
</div>
此时你可能已经达到了开启第二个角色的等级，会弹出让你创建第二个角色的提示。创建后你需要对新角色重新完成所有任务，此后你创建的所有角色也是这样。可以先不用管他，也可以创建1个新角色先让他砍树挂机。值得一提的是所有角色的物品箱共享，背包不共享。物品箱在城镇Anvil(制作台)的旁边。
<br />
然后任务会奖励你一个背包，打开背包，然后长按你刚刚获得的背包，就可以使用背包，这个背包的作用是增加你的背包容量，在后续你能获得很多背包，也是同样的使用方法。
值得一提的是，每个角色的背包不共享，背包容量也不共享，意思就是你每个角色都要去获得背包道具去提升背包容量，但对于前期的你并没有这么重要。
<br />
然后就是书卷蛇的下一个任务，它会要你分配五个技能点，点击下面的talents可以打开到天赋点页面:
<div style={{ textAlign: 'center' }}>

<div style={{ textAlign: 'center' }}>

<br />
回到正题，新手建议分配右上角的那个斧头，这个对伤害有直接加成。具体的加成方案可以在qq群里面查找加点方案。点完天赋，关闭页面，然后提交任务。
<br />
时间糖果：这个游戏的很多npc的任务会给你时间糖果，这个时间糖果可以让你在对应的动作中，比如打怪，砍树，挖矿等动作中直接获得x小时的afk收益。在背包中长按可使用。
<br />
书卷虫的下一个任务是让你去弄一个采矿证书，世界1主城下图箭头位置就是矿场的位置，点击传送门进去。
<div style={{ textAlign: 'center' }}>

</div>
来到这里后，点击左边的哥布林接任务，他是采矿证书的来源。第一个任务是让你挖铜矿，点击图中的堆在一起的东西，然后你的小人就会自己挖矿。
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/minemap1.png" alt="挖矿" style={{ width: '40%' }} />
</div>
采矿的时候上面有两个按钮，左边那个是通过小游戏来获得一段时间的afk收益，小游戏的收益由你的采矿效率和你的小游戏的分数决定，右边的是挂机。你切换到角色页面的时候你的小人也会挂机。
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/mine-minigame.png" alt="挖矿" style={{ width: '15%' }} />

挖了两个铜矿之后点击哥布林交任务，下一个任务哥布林教你去熔炼矿石，点击下方的熔炉，那个着火的东西，你会进到熔炉的界面:
<div style={{ textAlign: 'center' }}>

先别管我的熔炉框为啥是紫色的（这个以后我会讲）。最上边的四个是你的熔炉槽位页，到后面最多能有16个熔炉槽位，右上角的那个带箭头的是你的升级。中间左边的是放矿石的，中间是放燃料的地方，右边是产出。需要注意的是，烧矿不用燃料，燃料只会加快进度。最右边那个条状的东西是你的进度，左下角的tap是快速收矿，烧完矿后你点击矿锭就能直接放到背包，右下角是一键放矿石，把你背包里的矿石都放进熔炉里面烧。哥布林给你了一罐燃料，你可以用它熔炼矿石。等一会就好了，然后交任务。
<br />
哥布林的下一个任务是叫你升级熔炉，点开熔炉，然后点击右上角的黄色箭头：

点完两点完成这个任务，就能获得采矿证书啦。
<br />
这个是哥布林奖励给你的容量袋，长按使用，可以提升你单个格子可以容纳的矿石和矿锭的数量。之后的容量袋可以在打铁铺里面制造。

#### 背包页面讲解：
上图右下角是你的钱数，你的钱到达100的时候就会转换成下一个单位，比如100铜币可以自动转换成1个银币，以此类推。钱币左边那个锁头是背包锁，点击它，然后按格子，就可以把格子里的东西锁住，这样做的好处在于可以防止你快速放置的时候，把想留在背包里的东西丢进存储箱。右侧不同颜色的背包是你的小背包，点击可以跳转到不同的背包页面。中间的是你的装备，这个自己看。<br />
重点讲一下最左边的，左上角的是你的角色详情，会左边的页面会显示你的角色的等级，职业，和属性的大小，从上到下分别是，血量，蓝量，伤害，<span style={{ color: '#E83015' }}>力量，敏捷，智慧，运气，</span>暴击率，暴击伤害，<span style={{ color: '#E83015' }}>精准，防御，</span>移动速度，<span style={{ color: '#E83015' }}>掉率，</span>经验倍率，标红的内容是重点内容。
其中的精准度和防御：
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/accuracy-and-defense.png" alt="精准度和防御" style={{ width: '15%' }} />
</div>
精准度关乎于你打不打得到怪，防御则决定你会不会被怪打死。这两个属性都会影响你的打怪afk效率，需要达到要求的精准值才能打得到怪，但超过的部分没用。
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/afk-info.png" alt="afk信息" style={{ width: '48%' }} />
</div>
在打怪地图按右下角菜单，afk info，可以看到自己的挂机面板，如上图，其中3是无伤所需要的防御，这个和你的生存率有关，生存率会影响挂机效率，4是打这个怪所需要的精准，到达100%打怪，怪就不会miss，没到达100%打怪有概率miss，越低越容易miss。
<span style={{ color: '#E83015' }}>一定要注意精准度，精准度不够会影响你的打怪效率，从而降低你的挂机效率。</span>

</div>

<div style={{ textAlign: 'center' }}>

#### 存储箱：
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/chest.png" alt="存储箱" style={{ width: '40%' }} />
</div>
储存箱可以存放物品，右下角粉圈可以存放金钱，因为金钱不是角色共享的，得放到箱子里然后其他角色可以在箱子里取出来（中间下那个粉圈），左下那个粉圈是储存详细，最左边是你的储存的一些操作，从上到下依次是：<br />
第一个是快速储存，没点快速储存之前你存放物品只能一个一个拖进去，一个一个拖出来。按了之后可以点一下物品，然后物品就能出来。<br />
第二个是存放所有，点一下你背包里的物品都会存放在里面，除了背包里面带锁的格子的物品。<br />
第三个是分堆取出，按一下可以启用分堆模式，此时点击箱子里的物品会弹出这个界面：
<div style={{ textAlign: 'center' }}>
  <img src="/images/tutorial/stack-extract.png" alt="分堆取出" style={{ width: '18%' }} />
